인게임 화면의 핵심 결정 8가지

🟫 체크보드 = 웜 톤으로 교체

기존 다크(#1a1722/#13111a) → 웜 피치(#FFEFE0/#FFDFC8). v1~v3 그라데이션과 한 결. 가독성 잃지 않으면서 톤 통일.

📊 HUD = 4슬롯 + 진행바

TIME/SCORE/COMBO/PLAYER 4슬롯 유지. 시간 12초 이하면 슬롯 자체가 코랄 경고색으로 전환 + 얇은 진행바. "시간이 줄어든다"를 시각으로 즉각 인지.

🎵 음표 = 강조된 수집물

화이트 링 + 골드 글로우. 체크보드 바닥에서 시선이 자석처럼 끌리도록. 콤보 5 도달 시 ×5 팝업이 코랄+골드로 떠오름.

🔴 적(수간호사) = 권위 톤

플레이어보다 1.3배 크고, 흰 가운 + 회색 머리 + 사각 안경 + 찌푸린 눈썹. 머리 위 ! 버블이 깜빡거리며 위협 표현.

🔤 F 투사체 = 빨간 알약 + 트레일

"Fail" 학점이라는 컨셉. 코랄-레드 라운드 사각형에 "F" 텍스트 + 살짝 회전 + 후행 트레일 라인. 학생 트라우마 코미디.

🎮 D-Pad = 우하단 글래스 필

반투명 화이트 + 네이비 보더 4방향. 우하단 고정 (사용자 오른손 엄지). 중앙 데드존을 작은 다크 사각형으로 시각 표시.

🔘 스킬 버튼 = 좌하단 코랄 원형

좌하단 큰 원형 버튼 (GameConfig.skillExplanationControlHintText "좌하단 스킬 버튼"과 일치). 아이콘 + 우상단 B 키 라벨 + 아래 스킬명 칩. v1 시작 버튼과 동일 코랄 + 입체 그림자.

⏸️ Pause = 미니 우상단

다크 네이비 라운드 사각형 + 흰색 || 두 줄. 32px 크기로 작게 — 게임플레이 시선 방해 최소화.

HUD 상태 변화 (시간 임박 시)

00:45 ~ 00:16 (평상시)

다크 네이비 슬롯 + 골드 라벨. 진행바 골드. "여유 있음" 톤.

00:15 ~ 00:00 (긴급)

TIME 슬롯만 코랄로 전환(#C44A3D). 라벨/진행바 모두 크림색. 화면 가장자리도 약하게 펄스(미구현, 추천).

콤보 ≥ 3 (핫)

COMBO 슬롯 값만 골드(#FFB347)로. 콤보 ×5, ×10 마일스톤 도달 시 화면 중앙 팝업(현재 보이는 상태).

피격 (BREAK)

"BREAK" 라벨이 화면 가운데서 떨어지며 사라짐. COMBO 슬롯 값 → 0 + 코랄 플래시 1회.

난이도별 맵 변형 (같은 톤 유지)

EASY · 중앙 기둥 1개

가운데 단일 사각 기둥. 적 1마리. 시야 트인 학습용 맵.

NORMAL · 분리벽 + 장식 2개 (현재 표시)

중앙 세로벽(2칸 문) + 좌·우 2×2 장식 기둥. 정간호 돌진으로 분리벽 1칸 파괴 가능.

HARD · 4 코너 방 + 중앙 댄스플로어

네 모서리 작은 방 + 중앙 4개 기둥. 시야 차단 多. StoneGuard 이스터에그 진입 가능.
