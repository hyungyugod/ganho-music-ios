건간호 v6 — 완전 새 디자인 (쿠키런 톤)

  • ⭐ 큰 둥근 검은 눈 + 단일 화이트 하이라이트 (정석 chibi 공식)
  • ⭐ 둥글둥글한 표준 머리 비율 (rx 34, ry 36)
  • 부드러운 한 덩어리 앞머리 (jagged ❌, smooth ⭕)
  • 머리 위 작은 한 가닥 (귀여운 포인트)
  • 안경·혀·눈썹 디테일 다 제거 — 단순 화이트 하이라이트 + 작은 미소
  • 책 소품 + 라벤더 스크럽 (정체성 유지)

이간호 v3 — 곱슬 단발

  • 곱슬 단발 (턱선 길이) + 앞머리 + 옆머리 컬 디테일
  • 강아지 귀 ❌ 제거 (사용자 결정)
  • 둥근 미소 + 따뜻한 표정 (사진 얼굴 분위기)
  • 붉은 블러시 강조 (빨간 스웨터 톤 매핑)
  • 닫힌 눈 happy smile (사진의 따뜻함 강조)

임간호 v6 — 완전 새 디자인 (쿠키런 톤)

  • ⭐ 건간호와 동일한 큰 둥근 눈 + 단일 화이트 하이라이트 (시리즈 통일감)
  • ⭐ 회전·복잡한 디테일 다 제거
  • 풀어내린 부드러운 긴머리 + 중앙 가르마 앞머리
  • 작고 단순한 삼각형 고양이 귀 (정체성)
  • 작은 핑크 코 + 작은 미소
  • 수염·속눈썹·sparkle 다 ❌ — 무난한 톤 정석

정간호 마라톤 사진 기반

  • 둥근 검정 안경 (사진 시그니처) ⭐ 새 특징
  • 러닝캡 (사진 핑크 GO PINK PACE 모자)
  • 운동선수 같은 다부진 인상
  • 집중·결연한 눈썹 (마라톤 사진의 표정)
  • 땀방울 디테일 (운동 정체성)
  • 곡괭이 + 민트 스크럽 (태그 유지)
기존 정간호 디자인엔 안경이 없었는데, 사진에 안경 있어서 추가했어. 안 쓰는 게 좋으면 빼줄게.

✅ v6 확정 — 무난 귀엽 정석

  1. 김간호 → 번머리 + 헤드폰 (기존 유지)
  2. 정간호 → 안경 ⭕ + 핑크 러닝캡 + 곡괭이
  3. 건간호 v6 → 완전 새 디자인 · 큰 둥근 눈 + 단순 화이트 하이라이트 + 책
  4. 임간호 v6 → 완전 새 디자인 · 건간호와 동일 눈 톤 + 고양이 귀 + 긴머리
  5. 이간호 → 곱슬 단발 + 앞머리 (강아지귀 ❌)

📌 건간호 · 임간호는 큰 둥근 검은 눈 + 단일 화이트 하이라이트 정석 공식 적용. 복잡한 디테일(회전·속눈썹·수염·혀) 다 제거. 머리·소품으로만 차별화.
