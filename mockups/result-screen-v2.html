🎴 카드 1개 + 분기별 헤더

"GAME OVER" 같은 부정적 단어 대신 "실습 종료" / "NEW BEST!" / "수료 증서"로 3가지 톤을 분기. 카드 레이아웃은 동일.

📊 점수 디스플레이 = 코랄 → 골드

일반: 코랄 (#FF6B5B). 신기록: 골드 그라데이션 + 글로우 필터. 같은 64pt 숫자인데 색만 바뀌어서 임팩트.

🏆 BEST 라벨 = 칩 → 황금 칩

일반은 작은 골드 알약. 신기록일 땐 황금 그라데이션이 좌우로 흐르는 shimmer 애니메이션 + 그림자.

✨ NEW BEST 별 파편 5개

화면 곳곳에 ★ / ✦ 5개. 펄스 + 회전 애니. heavy 햅틱 + NewMail 사운드와 동시 발화 (기존 코드 유지).

📜 졸업장 = 우드컷 종이 톤

대각선 패턴 + double-border + 황토 컬러로 진짜 증서 느낌. 한글 명조(Gowun Batang) 폰트 + 더블 모서리 장식 + 빨간 도장.

🗣️ 분기별 부제 카피

A: "수고했어요! 한 번 더 해볼까요?" / B: "최고 기록을 갱신했어요!" / C: 본문 2줄 (DiplomaOverlayNode 코드 그대로). 톤이 점수 안 짠 게임에서 사용자를 응원함.
