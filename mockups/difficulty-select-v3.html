Phase C 핵심 결정 (v2 → v3)

🎨 하·중·상 색만으로 강도 즉시 인지

v2까지는 3장 카드가 동일 피치 톤이라 "왼쪽이 쉬운지 어려운지"를 라벨로만 알 수 있었다. v3는 카드 배경 자체에 위계 — 하=민트(#9BE0CC→#5EBFA3), 중=골드(#FFD27A→#E5A647), 상=코랄(#FF8E80→#FF6B5B). 0.5초 안에 강도 인지.

✨ 선택 글로우 80% · 미선택 0.78 — 시선 자석

선택 카드는 카드별 색 라디얼 글로우 158×116 + α 0.80 + blur 20px로 화면에서 가장 강한 시선 자석이 된다. 미선택 두 장은 opacity 0.78로 자연스럽게 물러난다. 위계가 *자동으로* 보임.

▶ 시작 버튼 halo = 마지막 결정

입체 그림자 6 → 8px로 묵직함 강화. 뒤에 240×90 코랄 halo가 0.25s 페이드 인 — "이게 마지막 탭"이라는 시각 약속. PrimaryButton 내부 0 변경(다른 화면 회귀 위험 0).

🛠 SpriteKit 매핑 — Difficulty 4 lookup + Scene halo SKShape

Difficulty에 cardFillTop/cardFillBottom/cardStrokeColor/cardGlowColor 4 computed property 추가(3 case exhaustive). DifficultyCardNode init/setSelected에서 id.color → id.cardFillTop/cardStrokeColor 분기. liftCurrentOffset 증분 패턴. halo는 DifficultySelectScene이 외부 SKShapeNode 직접 부착 — PrimaryButtonNode 내부 0줄.
