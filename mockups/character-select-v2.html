🎴 카드 = 글래스 + 색 점

5장 모두 반투명 화이트 카드로 톤 통일. 각 캐릭터 정체성은 우상단 색 점(8px)으로만 표현 — 카드 자체 색을 다르게 하면 시각 노이즈가 됨.

✨ 선택 카드 = 떠오르기

translateY(-12px) + scale(1.08) + 코랄 보더 + "선택됨" 뱃지. 5장 중 어느 게 골라졌는지 시선이 바로 잡힘.

👤 아바타 차별화

모두 같은 베이스(둥근 얼굴 + 간호사 모자)에서 머리/액세서리만 변주. 시각 일관성 + 빠른 식별. 김간호=번머리, 정간호=헤어밴드+곡괭이, 건간호=안경+책, 임간호=고양이귀+긴머리, 이간호=강아지귀+단발.

↩️ Back = 메인으로 (Sprint 6)

난이도가 흐름 마지막으로 이동하면서 캐릭터 선택 위에 더는 "직전 단계 = 난이도"가 없다. 백버튼은 곧장 메인(StartScene)으로 돌아간다.

📋 스킬 정보 패널

선택된 캐릭터 정보를 화면 하단 가운데에 표시. "스킬 이름" + "속도 배율" 2가지 핵심만. 다크 네이비로 BG와 분리해 가독성 확보.

↩️ Back 버튼 = 글래스 필

화살표 + 한 줄 텍스트. 메인 화면의 통계 칩과 동일한 글래스 필 패턴 재사용 — 학습 부담 없음.
