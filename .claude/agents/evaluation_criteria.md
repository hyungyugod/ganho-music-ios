## AI 슬롭 패턴 — 자동 감점 목록

다음 패턴 발견 시 해당 항목에서 -1점 적용.

| 패턴 | 감점 항목 | 이유 |
|---|---|---|
| 강제 언래핑 `!` 남발 | 성능 & 안정성 | 크래시 원인 |
| `Timer.scheduledTimer` 사용 | Swift 패턴 | SKAction으로 대체해야 함 |
| `update()` 안에 `addChild()` 반복 | 성능 & 안정성 | 매 프레임 노드 생성 = 성능 저하 |
| 매직 넘버 하드코딩 (`speed = 200.0` 등) | Swift 패턴 | GameConfig 상수 미사용 |
| SPEC에 없는 독립 기능 추가 | 기능 완성도 | 범위 위반 |
| 고정값 이동 (`position.x += 5`) | 성능 & 안정성 | dt 기반 이동 미적용 |
| 클로저 내 `self` 강한 캡처 | 성능 & 안정성 | 메모리 누수 |

## 평가 기준 (4개 항목)

### 1. Swift 패턴 일관성 (비중: 35%)

기존 Swift 규칙과 코딩 컨벤션을 따르는가?

합격: guard let 사용, MARK 섹션 구분, GameConfig 상수, 함수 단일 책임, 적절한 네이밍
불합격: 강제 언래핑, 매직 넘버, 함수 비대화, 네이밍 불일치

**점수 기준선**
- 8점: 모든 패턴 준수. P2 수준 사소한 불일치 1~2건.
- 5점: 강제 언래핑 1건 이상, 또는 매직 넘버 3건 이상.

### 2. 게임 로직 완성도 (비중: 30%)

SpriteKit 패턴을 올바르게 사용하고 게임 로직이 의도대로 작동하는가?

합격: didMove(to:) 초기화, dt 기반 이동, SKAction 스폰, PhysicsCategory 비트마스크, GameState 관리
불합격: Timer 사용, 고정값 이동, 충돌 처리 내 즉시 삭제, 상태 관리 없음

**점수 기준선**
- 8점: SpriteKit 패턴 준수. 게임 루프 정상 작동. 사소한 최적화 여지 있음.
- 5점: Timer 사용 또는 dt 기반 이동 미적용.

### 3. 성능 & 안정성 (비중: 20%)

크래시 없이 안정적으로 작동하고 성능 문제가 없는가?

합격: 강제 언래핑 없음, weak self 캡처, 노드 정리 적절, 빌드 에러 없음
불합격: 크래시 원인 존재, 메모리 누수, update() 과부하

**점수 기준선**
- 8점: 크래시 원인 없음. 빌드 클린. weak self 적용.
- 5점: 강제 언래핑 1건 이상 (P0).

### 4. 기능 완성도 (비중: 15%)

SPEC.md에 명시된 기능이 모두 구현되었는가?

합격: 모든 SPEC 기능 구현, 게임 상태 전환 정상, 예외 처리 포함
불합격: SPEC 기능 누락, 미완성 코드, 초기화 미등록

## 판정

가중 점수 = (Swift패턴×0.35) + (게임로직×0.30) + (성능안정성×0.20) + (기능완성도×0.15)

- **7.0 이상** → 합격
- **5.0~6.9** → 조건부 합격 (피드백 반영 후 재검수)
- **5.0 미만** → 불합격
- **P0 이슈 1건 이상** → 점수 무관 불합격
